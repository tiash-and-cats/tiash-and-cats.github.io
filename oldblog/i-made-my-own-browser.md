# I made my own browser!

<i>May 22, 2025 <small class="edited">edited May 23, 2025</small></i>

<p>It's called <strong>AirSquirrel</strong>, and you can install it by clicking 
   <a href="https://tiashdev.github.io/tiashfam-resources/AirSquirrel_Installer.exe" download>here</a>.
</p>

<p>AirSquirrel is a lightweight web browser built with WebView2, designed for fast and smooth browsing.</p>
<p>Currently, it is only available for Windows.</p>

<h2>Installation Instructions</h2>
<p>Download <code>AirSquirrel_Installer.exe</code> using the link above.</p>
<p>If Windows warns you about a potential threat, whitelist the installer in your antivirus—it's completely safe.</p>
<p>Right-click the installer and select <strong>"Run as administrator"</strong> <em>(Required only for system-wide installation)</em>. Click "Yes" when asked.</p>
<p>A black screen with text will appear to guide you through the installation:</p>
<ul>
   <li>Choose the installation type:
      <ul>
         <li>Enter <strong>1</strong> to install for your user only (Recommended).<br>  
             <b>Default install folder:</b> <code>C:\Users\<i>YourName</i>\AppData\Local\Programs\AirSquirrel</code>.  
             <em>(No administrator privileges required.)</em>
         </li>
         <li>Enter <strong>2</strong> to install for everyone on the computer.<br>  
             <b>Default install folder:</b> <code>C:\Program Files (x86)\AirSquirrel</code>.  
             <em>(Administrator privileges required.)</em>
         </li>
      </ul>
   </li>
   <li>Confirm the installation folder (or press <strong>ENTER</strong> to use the default).</li>
   <li>AirSquirrel will install and appear in your apps.</li>
   <li>Choose whether to:
      <ul>
         <li>Add a shortcut to your desktop (Press Y for Yes and ENTER, or N for No and ENTER).</li>
         <li>Open AirSquirrel now (Press Y for Yes and ENTER, or N for No and ENTER).</li>
      </ul>
   </li>
</ul>

<h2>Uninstallation Instructions</h2>
<p>To remove AirSquirrel:</p>
<ol>
   <li>Open the <strong>Start Menu</strong> and find AirSquirrel.</li>
   <li>Right-click it and select <strong>"Uninstall"</strong>.</li>
   <li>A window will appear. If AirSquirrel isn't already selected (i.e. with a blue box around it), click on it.</li>
   <li>Click <strong>"Uninstall/Change"</strong> and follow the steps.</li>
</ol>
<p>The uninstaller removes most files automatically, but if the installation folder remains, manually delete:</p>
<ul>
   <li>If installed for your user only: <code>C:\Users\<i>YourName</i>\AppData\Local\Programs\AirSquirrel</code>.  
       (Replace <i>YourName</i> with your actual Windows username.)
   </li>
   <li>If installed system-wide: <code>C:\Program Files (x86)\AirSquirrel</code>.</li>
   <li>Or, if you selected a custom install folder, delete that.</li>
</ul>
<p><strong>Optional cleanup:</strong> To ensure all leftover files are removed, you can use third-party software like <strong>IObit Uninstaller</strong> or <strong>Revo Uninstaller</strong> to scan for and delete any remaining AirSquirrel data.</p>
